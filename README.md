# TayyabExpress - Modern E-Commerce Platform

TayyabExpress is a complete, full-stack e-commerce web application built using the MERN stack (MongoDB, Express, React, Node.js). It is designed to provide a modern, seamless shopping experience with features like real-time search, secure authentication, and a responsive mobile-friendly UI.

### üõí Shopping Experience
* **Dynamic Product Catalog:** Browse hundreds of unique products across 8 categories.
* **Search & Filtering:** Real-time product search and category filtering.
* **Product Details:** High-quality images, specifications, ratings, and reviews.
* **Smart Image Fallback:** Automatically replaces broken images with a placeholder to keep the UI clean.
* **Shopping Cart:** Add to cart, adjust quantities, and view live subtotal calculations.

* **Order History:** Track past purchases with detailed breakdowns.
